Git 기초#

- .git 폴더는 무엇인가요?
->  커밋이 담겨져있는 폴더
- .git 폴더를 만드는 방법은?
-> git init

Project 폴더 -> 내가 작업할 최상위 폴더(root directory)

git 도움을 주는 확장앱 설치
-> Git Graph

-1 gitignore

-2 Reset, Revert 커밋을 뒤로 돌아가는 행위

-3 branch  커밋을 나누는 행위

-4 merge, rebase 커밋을 합쳐주는 행위

-5 git log 명령어 
-> 커밋했던 히스토리를 볼수있다.
